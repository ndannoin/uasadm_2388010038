01 — Tentang Saya

Siapa Saya?

Saya mahasiswa Informatika di UIN Siber Syekh Nurjati Cirebon yang memiliki ketertarikan mendalam pada teknologi kecerdasan buatan, khususnya implementasi Object Detection menggunakan framework YOLO dan library OpenCV. Saya terbiasa mengolah dataset, melakukan labeling lewat Label Studio, serta melatih model cerdas berbasis data visual maupun tekstual.

Menguasai pemrograman Python serta manajemen environment pengembangan modern seperti Anaconda. Di samping fokus riset teknis, saya juga menikmati eksplorasi teknologi server lokal dan analisis ekosistem digital demi melahirkan solusi mutakhir yang solutif.

Saya percaya bahwa Computer Vision dan kecerdasan buatan adalah jembatan utama untuk mempermudah otomasi industri serta menciptakan dampak nyata di masyarakat global.
